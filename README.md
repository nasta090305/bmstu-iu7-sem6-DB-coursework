# Курсовая работа по курсу "Базы данных" на тему "Разработка базы данных для системы мониторинга оборудования на тракторном заводе" МГТУ ИУ7. 6 семестр, весна 2025.

### Научный руководитель: Дроздецкая П.В.

## Техническое задание

Провести анализ предметной области, определить основные сущности и их взаимосвязи. Сформулировать требования и ограничения к разрабатываемой базе данных. Спроектировать сущности базы данных и определить ограничения целостности данных. Спроектировать ролевую модель на уровне базы данных. Выбрать средства реализации базы данных и приложения. Разработать и реализовать сущности базы данных и ограничения целостности. Описать методы тестирования разработанного функционала и разработать тесты для проверки корректности работы приложения. Провести исследование зависимости времени выполнения запросов к базе данных от объема обрабатываемых данных.

## Цель

Целью данной курсовой работы является разработка базы данных для системы мониторинга оборудования на тракторном заводе, которая позволит централизованно собирать, хранить и анализировать данные о состоянии техники.

## Задачи

- Провести анализ предметной области с целью выявления основных объектов и процессов, связанных с мониторингом оборудования на тракторном заводе.
- Определить ключевые сущности базы данных и их взаимосвязи, необходимые для эффективного представления данных в рамках предметной области.
- Сформулировать требования и ограничения, предъявляемые к проектируемой базе данных, включая функциональные и нефункциональные аспекты.
- Спроектировать структуру базы данных, включая описание сущностей, атрибутов, связей между сущностями и ограничений целостности данных.
- Разработать ролевую модель безопасности на уровне базы данных для разграничения прав доступа к данным и обеспечения их защиты.
- Выбрать подходящие средства реализации базы данных и приложения, исходя из специфики задачи и доступных технологий.
- Реализовать спроектированную базу данных, включая создание таблиц, ограничений и других объектов базы данных.
- Разработать методы тестирования и реализовать тесты, направленные на проверку корректности функционирования базы данных и приложения.
- Провести исследование производительности, изучив зависимость времени выполнения запросов от объема обрабатываемых данных, и сделать выводы по результатам анализа.
